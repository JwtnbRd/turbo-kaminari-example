プロジェクトの初期設定として技術スタックの選定を行います。

## 実行内容

ユーザとのインタラクティブなやりとりを通じて、プロジェクトに最適な技術スタック定義を作成し、`docs/tech-stack.md` ファイルを生成します。

### Phase 1: プロジェクト基本情報のヒアリング

まず、以下の質問にお答えください：

**質問1: プロジェクトタイプ**
以下から最も近いものを選択してください：

1. **Webアプリケーション** - ブラウザで動作するアプリケーション
2. **モバイルアプリ** - スマートフォン/タブレット向けアプリ
3. **デスクトップアプリ** - PC向けアプリケーション
4. **API/バックエンドサービス** - データ処理・API提供
5. **CLI/コマンドラインツール** - ターミナルで動作するツール
6. **ライブラリ/フレームワーク** - 他の開発者が使用するコンポーネント
7. **その他** - 上記に当てはまらない場合

**質問2: 主な用途・目的**
プロジェクトの主な目的を教えてください：

1. **業務効率化** - 社内業務の自動化・効率化
2. **顧客向けサービス** - エンドユーザー向けのサービス提供
3. **学習・実験** - 技術習得や概念検証
4. **データ処理・分析** - 大量データの処理・可視化
5. **その他** - 具体的な目的を自由記述

まずこれらの質問にお答えいただいた後、詳細な技術選択のヒアリングを行い、最終的に `docs/tech-stack.md` を生成いたします。
